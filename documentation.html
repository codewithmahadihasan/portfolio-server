API Documentation

Base URL

localhost:5001/api/v1

Authentication

This API requires authentication. For the POST and PUT methods, you need to include a specific header value mohotasim_hadi_rafi.

Authorization: mohotasim_hadi_rafi

Endpoints

Create Resource

URL: POST /resource

Description: Create a new resource.

Headers: Authorization: mohotasim_hadi_rafi

POST /api/v1/resource

            Request Body:
            {
                "name": "Resource Name",
                "description": "Resource Description"
            }

Update Resource

URL: PUT /resource/{id}

Description: Update an existing resource.

Headers: Authorization: mohotasim_hadi_rafi

PUT /api/v1/resource/123

            Request Body:
            {
                "name": "Updated Resource Name",
                "description": "Updated Resource Description"
            }
